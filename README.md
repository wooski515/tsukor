# TSUKOR

Онлайн-галерея моєї колекції цукрових пакетиків — цукор, сіль та перець.

Статичний сайт без залежностей: HTML + CSS + JavaScript. Задеплоєний на GitHub Pages.

## Структура

```
index.html   — сторінка галереї
images.js    — список фото для кожної категорії
style.css    — стилі
img/tsukor/  — фото пакетиків цукру
img/sil/     — фото пакетиків солі
img/perets/  — фото пакетиків перцю
```

## Як додати нове фото

1. Покладіть файл у папку потрібної категорії, наприклад `img/sil/`.
2. Допишіть назву файлу у відповідний список в `images.js`:

```js
sil: [
    "1.png",
    "нове фото.png",
],
```

Фото показуються у тому порядку, в якому записані у списку.

## Можливості

- Вкладки-категорії: Цукор / Сіль / Перець
- Лупа для розглядання пакетиків зблизька (клік по фото, на ПК)
- Кнопки «+» / «−» для зміни розміру фото
- Адаптивна версія для телефонів
